# craftweg.com

Craftweg's website powered by [Zola](https://www.getzola.org/)

## Set up

1. Clone the repository: `git clone https://github.com/craftweg/website.git`
2. Install [Zola](https://www.getzola.org/)
3. Run `zola serve`.
